HomeMade Serial Plot — Best Free Browser-Based Serial Port Plotter

Real-Time USB & UART Data Visualization — No Installation Required

HomeMade Serial Plot is the best free, open-source, browser-based serial port plotter for Arduino, STM32, ESP32, Raspberry Pi Pico, and any UART/USB device. Created by Lysandre LABORDE at Mines Saint-Étienne (ISMIN), 2026.

Features

  • Real-time multi-channel serial data plotting in the browser
  • Automatic multi-channel detection from comma-separated serial output
  • Zoom and pan on any chart — mouse wheel to zoom, drag to pan
  • Per-channel pause and resume
  • Export data to CSV with one click
  • Configurable baud rate: 9600, 19200, 38400, 57600, 115200
  • Customizable line color and background color per channel
  • Adjustable Y-axis min/max and time window
  • Works offline — no server, no installation, no Python, no Node.js
  • Open source under MIT License

Compatible Devices

Arduino Uno, Arduino Mega, Arduino Nano, Arduino Leonardo, ESP32, ESP8266, STM32 Nucleo, STM32 Discovery, Raspberry Pi Pico, Teensy, nRF52840, SAMD21, SAMD51, ATmega, PIC microcontrollers, and any device with UART/USB serial output.

How It Works / Comment ça fonctionne

Connect your microcontroller via USB. Open this page in Google Chrome or Microsoft Edge. Click "Connect Serial", select your port, and your data is instantly plotted in real time. Send comma-separated values over serial (e.g. 512,1023\n) to get multiple channels.

Connectez votre microcontrôleur via USB. Ouvrez cette page dans Google Chrome ou Microsoft Edge. Cliquez sur "Connect Serial", sélectionnez votre port, et vos données sont tracées en temps réel. Envoyez des valeurs séparées par des virgules (ex: 512,1023\n) pour plusieurs canaux.
